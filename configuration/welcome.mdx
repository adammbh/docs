## Creating a welcome channel


<Steps>
  <Step title="Head over to your Discord Server">
    Click on the Discord Server that you would like to setup welcome for using
    Evict.
  </Step>
  <Step title="Create the channel">
    Setup permissions for the channel with the desired name, for this example we
    will be using `#welcome`
  </Step>
</Steps>

## Creating an embed and setting the welcome channel

<Steps>
  <Step title="Creating the welcome embed">
    Run the `,embedsetup` command while to create the welcome embed which you
    can use variables with, for more information regarding variables:
    [/configuration/variables](/configuration/variables)
  </Step>
  <Step title="Setup the welcome channel">
    Run the `,welcome add` command which will give you further information, for this
    example we'll run: `,welcome add #welcome {title: Welcome!} {author: {user.name}}`
  </Step>
</Steps>
